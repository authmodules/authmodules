# AuthModules

AuthModules is organized around a minimal core-first baseline. This repository is the umbrella repository for cross-repository planning, governance, and public coordination.

The first package repository is `authmodules/core`, published as `@authmodules/core`. Package repositories are created only when there is a tracked Project item and a scoped issue with an actual implementation or documentation need.

## Governance

Repository governance is defined in these documents:

- [Repository governance baseline](docs/repository-governance.md)
- [New repository checklist](docs/new-repository-checklist.md)
- [Pull request review policy](docs/pr-review-policy.md)
- [GitHub Project workflow](docs/projects.md)

Current governance work is documentation-only. It does not change repository settings, add automation workflows, or modify domain code.
